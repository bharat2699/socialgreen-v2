Welcome to SocialGreen

THINK GREEN, BE GREEN!

How to use :

Hey, Planters how is life going, let me introduce you with some basic nomenclature

  • PLANTER : User of this app
  • LEAF : Post in this app
  • LEAFS : Plural of post in this app
  • SEED : Personal message in this app
  • SEEDS : Plural of personal message in this app

You can access each and every feature of this app by Logging in, if you are not logged in SignUp now, Now, you will see all the features in that NavBar.

FOR BETTER EXPERIENCE, BROWSE ON FULL SCREEN.

Some Awesome Features :

  • SignUp functionality.
  • Login and Logout functionality.
  • Posting Text updates.
  • Feed page will have posts by you as well as the planters you follow.
  • Search for other Planters as well as there Leafs.
  • "Dont be Shy just say Hi!", to your fellow planters through Chats.
  • "Love all, Rival none", only LIKE button enabled.
  • Edit your profile and add some cool pic of yours.
  • visit planters profile from anywhere you want.
  • See the name of planters who follow you in profile.
  • Other features are Easter Eggs explore more to see.
